== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install King Composer plugin then install Ultimate Shortcodes Addon for King Composer.

This addon plugin works like other page builder addon plugin. Go to new page then click on King Composer builder then click on Add Element you should see Ultimate Shortcodes Addon menu on right, click on it you will see Hover Effects shortcode button. Open it and browse an image and customize according to your needs.
